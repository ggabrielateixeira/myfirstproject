HTML BÁSICO

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página Web e seu conteúdo. Basicamente, consiste em uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.

Em outras palavras, ao acessar uma página web através de um navegador, ele é capaz de interpretar o código HTML e renderiza-lo de forma compreensível para o usuário final, exibindo textos, botões e outros com as configurações definidas por meio das diversas tags que essa linguagem dispõe.

SURGIMENTO DA INTERNET

Redirecionar ao início da página

A Internet teve início como um projeto militar nos Estados Unidos, chamado ARPANET, na década de 1960. Seu objetivo era criar uma rede de comunicação segura entre instituições de pesquisa. Com o tempo, evoluiu para uma rede global, interligando universidades, organizações e, eventualmente, o público em geral.

A World Wide Web (WWW), criada pelo cientista britânico Tim Berners-Lee em 1989, foi um marco fundamental ao permitir a navegação fácil e a troca de informações através de páginas da web. Desde então, a Internet se expandiu exponencialmente, tornando-se uma parte essencial da vida cotidiana para bilhões de pessoas em todo o mundo.

O surgimento da Internet foi resultado de contribuições de várias pessoas ao longo do tempo. Aqui estão algumas das pessoas mais importantes para o desenvolvimento da Internet:

  1. J.C.R. Licklider: Foi um dos pioneiros que concebeu a ideia de uma "rede galáctica" de computadores interligados e interativos na década de 1960.
  2. Leonard Kleinrock: Realizou pesquisas fundamentais sobre teoria de filas e transmitiu a ideia da troca de pacotes de dados, essencial para a comunicação na Internet.
  3. Vinton Cerf e Bob Kahn: São considerados os "pais da Internet". Desenvolveram o TCP/IP, o protocolo fundamental que permite a comunicação na Internet.
  4. Tim Berners-Lee: Inventor da World Wide Web (WWW) em 1989, criando o protocolo HTTP e o primeiro navegador web, o WorldWideWeb.

Essas figuras desempenharam papéis cruciais no desenvolvimento de tecnologias e conceitos fundamentais que possibilitaram a criação e a expansão da Internet.

COMO FUNCIONA O HTML

Redirecionar ao início da página

HTML é uma linguagem de marcação que permite criar documentos que podem ser visualizados em navegadores da web. Ela utiliza uma estrutura hierárquica composta de elementos, sendo que cada elemento é definido por meio de uma "tag". As tags são colocadas dentro de colchetes angulares, como <tag>, para representar elementos específicos.

A estrutura básica de um documento HTML inclui as tags <html> (define o início e o fim do documento), <head> (contém metadados e links para estilos) e <body> (contém o conteúdo visível da página).

Os elementos HTML são usados para estruturar o conteúdo, como cabeçalhos <h1>, parágrafos <p>, listas <ul> ou <ol> entre outros. As tags podem conter atributos, que fornecem informações adicionais sobre os elementos, como o caminho de uma imagem ou o endereço de um link.

Além disso, HTML permite criar links (<a>) para conectar páginas e incorporar imagens (<img>) para enriquecer o conteúdo visual. Também possibilita a criação de formulários (<form>) para coletar informações do usuário, como nomes e endereços de e-mail.

No geral, HTML é a base fundamental para a criação de páginas da web, sendo crucial para a estruturação, apresentação e interatividade dessas páginas na internet.

TAGS DO HTML

Redirecionar ao início da página

Um documento HTML é composto por TAGS, as quais possuem um nome e aparecem entre os sinais "<" e ">".

Essas tags devem ser abertas e fechadas, por exemplo: < body > < /body >. Nesse caso, a tag de fechamento deve conter a barra / antes do nome.

CABEÇALHO NO HTML

Redirecionar ao início da página

Os elementos de cabeçalhos permitem especificar que certas partes do seu conteúdo são títulos ou subtítulos. Da mesma forma que um livro tem o título principal e os capítulos possuem títulos e subtítulos, um documento HTML também tem. HTML contém 6 níveis de título, <h1> (en-US) - <h6> (en-US), embora você normalmente só use de 3 a 4:

  • <h1>: Utilizado para o título principal.
  • <h2>: Utilizado para títulos de alto nível.
  • <h3>: Utilizado para subtítulos.
  • <h4>: Utilizado para o segundo subtítulo.

PARÁGRAFOS NO HTML

Redirecionar ao início da página

O elemento <p> representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco.

Já em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.

IMAGENS NO HTML

Redirecionar ao início da página

Uma parte essencial do design e do conteúdo de uma página da web é a inclusão de imagens. As imagens são usadas para transmitir informações visualmente e melhorar a estética de uma página. Elas podem variar de fotografias a ícones e gráficos, desempenhando um papel vital na comunicação e na interação do usuário.

Em HTML, para incorporar uma imagem em uma página, usamos a tag <img>. Esta tag é uma tag de autotérmino, o que significa que não possui uma tag de fechamento. A <img> é uma tag bastante versátil, permitindo que especificemos a origem (URL ou caminho local) da imagem e forneçamos um texto alternativo descritivo.

CAMINHO DA IMAGEM

O atributo src (source) da tag <img> contém o caminho da imagem que queremos exibir. O caminho precisa estar correto para que o navegador possa carregar e exibir a imagem. Se a imagem não for encontrada, o navegador exibirá o texto alternativo especificado no atributo alt.

Vejamos um exemplo a seguir:

  • <href> (hipertexto de referência): Este é o atributo obrigatório que especifica o destino do link, que pode ser um URL, um caminho para um arquivo ou um identificador interno.
